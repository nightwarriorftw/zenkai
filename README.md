## :nut_and_bolt: Development


#### 1. Clone the Repository

```Bash
git clone https://github.com/nightwarriorftw/zenkai.git
cd zenkai
```

#### 2. Install the dependencies

```BASH
npm install
```

#### 3. Run server:

```BASH
npm start
```
